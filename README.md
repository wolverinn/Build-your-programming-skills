# Build-your-programming-skills
A learning road map for junior programmers focusing from basic to advanced technical skills

工作之后，应该如何规划自己的技术学习路线？可以学的太多，但我们不能什么都学，而是应该有一个清晰的roadmap，一步一步慢慢积累，这样才能获得最大的加速度
